Welcome!

Hey hey hey! Welcome to the offline editable release of the tutorial!

You can study it on Cook islands being completely out of the net, and everything would work. You can edit it and push updates which get merged in.

To start browsing, go to the language folder and open nav.html.

TroubleShooting

For Chrome/Chromium: run the browser with the flag --allow-file-access-from-files. For other issues and suggestions, use Github at https://github.com/iliakan/javascript-book/.

Contribution

Help us to create a really great JavaScript book!

You are welcome to:
  • Fix typos.
  • Translate the book.
  • Rephrase and improve the text, make it better.
  • Change styles, improve look and feel.
  • Add your chapters.

The contributions are carefully reviewed and appended or integrated with the current content.

All changes are done offline. This way is chosen, because it gives full freedom.

You can add images, HTML/JS/CSS examples and manage the text in your favorite editor.

The formatting of the text is HTML-compatible, with many features borrowed from Markdown and additional, powerful abilities for interactive JS and HTML/CSS. Read about it in about/format.html.

Sending changes

First, if you contribute something new, like translation or a new chapter, you can just send it in the way you like. See the email below.

If you know Git, then create a pull request for https://github.com/iliakan/javascript-book/.

If you don't know Git... Things are still easy. You can install and use it in 5 mins.

  1. Grab it from http://git-scm.com/, install.
  2. Run git clone git://github.com/iliakan/javascript-book.git jsbook, that creates jsbook folder with the book.
  3. Edit the book in that folder.
  4. To summarize changes, run git diff >my.diff in the folder. It saves all changes in the file my.diff.
  5. Send it on the email iliakan@javascript.info. Now they will get merged in.

Actually, Git is much more powerful than that. But there are more thorough books and manuals about it ;).
